redis数据类型

redis数据类型

String

String是redis最基本的类型，value 不仅可以是 String,也可以是数字。redis中的string类型是以二进制存储的，所以存储的内容也可以是图片等各个类型的文件

Hash

Redis hash 是一个string类型的field和value的映射表，hash特别适合用于存储对象。

List

List是简单的字符串列表，按照插入顺序排序。你可以添加一个元素导列表的头部（左边）或者尾部

输出结果

定义集合a，b

sadd a 1 2 3
sadd b 3 4 5

显示集合a的元素

smember a//1 2 3
smember b//3 4 5

求交集

sinter a b//3

求差

sdiff a b //1 2

添加到Sorted Set

zadd score 10 player1
zadd score 20 player2
zadd score 20 player3

升序显示Sorted Set中的内容

zrange score 0 -1 [withscores]

降序显示Sorted Set中的内容

zrevrange score 0 -1 [withscores]

MongoDB运维相关

数据的备份

用法：mongodump -h 主机IP：端口 -d 数据库名称 -o 备份文件所在的目录

数据的恢复

恢复整库用法：mongorestore -h 主机IP：端口 -d 数据库名称 –dir 备份文件所在目录

恢复一个集合的用法：mongorestore -h 主机IP：端口 -d 数据库名称 –dir 备份文件 目录下集合.bjson

数据导出

参数说明：

  • -d 数据库名
  • -c collection名
  • -o 输出的文件名
  • –type 输出的格式，默认为json
  • -f 输出的字段，如果–type为csv，则需要加上 - -f “字段名”

数据导入

参数说明

  • -d 数据库名
  • -c collection名
  • –type 导入的格式,默认json
  • -f 导入的字段名
  • –headerline 如果导入的格式是csv，则可以使用第一行的标题作为导入的字段
  • –file 要导入的文件
